Latest worldwide earthquakes

App
May 2021

About

It is an interactive map showing the latest worldwide earthquakes with the user interface. The UI contains a couple of functionalities like filtering by magnitude, by name and counting earthquakes in the country. Additionally, there are displayed basic information on each earthquake that the user clicks on.
The aim of this project was to get familiarized with the Leaflet library and keep on practising with JavaScript.

Technology

HTML, CSS, JavaScript, Leaflet
